# API Reference

Base URL (local): `http://localhost:5000/api`

All responses are JSON. All endpoints are read-only except `/predict`,
which accepts an image upload.

---

## `POST /api/predict`

Run a prediction on an uploaded fundus image.

**Request:** `multipart/form-data` with a single field `image`.

**Query parameters:**

| Param | Default | Description |
|---|---|---|
| `report` | `false` | If `true`, also generates a PDF report and returns its path. |

**Example:**

```bash
curl -X POST "http://localhost:5000/api/predict?report=true" \
  -F "image=@fundus_photo.jpg"
```

**Success response (200):**

```json
{
  "success": true,
  "prediction": {
    "predicted_class": "Diabetic_Retinopathy",
    "confidence": 0.8734,
    "top_k": [
      {"class": "Diabetic_Retinopathy", "confidence": 0.8734},
      {"class": "Healthy", "confidence": 0.0821},
      {"class": "AMD", "confidence": 0.0245}
    ],
    "all_probabilities": {"...": "..."},
    "risk_level": "High",
    "explanation": "The model detected visual patterns consistent with ...",
    "inference_time_ms": 452.3,
    "model_version": "1.0.0",
    "architecture": "EfficientNetB7",
    "timestamp": "2026-07-07T10:00:00+00:00",
    "gradcam_base64": "iVBORw0KGgoAAAANSUhEUgAA...",
    "disclaimer": "This prediction is generated by an AI research model...",
    "errors": []
  },
  "uploaded_image_path": "/absolute/path/to/uploads/<uuid>.jpg",
  "report_path": "/absolute/path/to/reports/report_<uuid>.pdf"
}
```

**Error response (400):**

```json
{
  "success": false,
  "errors": ["Unsupported file type '.gif'. Allowed: bmp, jpeg, jpg, png, tif, tiff"]
}
```

## `GET /api/health`

```json
{
  "status": "ok",
  "model_loaded": true,
  "num_classes": 5,
  "timestamp": "2026-07-07T10:00:00+00:00"
}
```

Used as Render's `healthCheckPath`.

## `GET /api/metrics`

Returns the most recent `evaluate.py` output.

**Before running `evaluate.py` (404):**

```json
{
  "available": false,
  "message": "No evaluation report found. Run `python evaluate.py` after training."
}
```

**After running `evaluate.py` (200):**

```json
{
  "available": true,
  "metrics": {
    "num_test_samples": 240,
    "class_names": ["AMD", "Cataract", "Diabetic_Retinopathy", "Glaucoma", "Healthy"],
    "classification_report": { "...": "sklearn classification_report as a dict" },
    "overall_accuracy": 0.912
  }
}
```
