Trained on five conditions

The classes the model screens for

Class list is discovered automatically from the dataset folder structure — this reflects the default configuration described in the project README.

Healthy

No signs consistent with the trained disease classes.

Diabetic Retinopathy

Microaneurysms, hemorrhages, and vessel changes from diabetes-related damage.

Glaucoma

Optic disc cupping and nerve fiber layer changes associated with elevated eye pressure.

AMD

Drusen deposits and macular changes seen in age-related macular degeneration.

Cataract

Lens clouding that reduces fundus image clarity and sharpness.

How it works

From upload to explanation, in one pass

Capture

Upload a fundus photograph — JPG, PNG, or TIFF, up to 10 MB.

Enhance

The image is denoised, contrast-enhanced with CLAHE, and cropped to remove vignetting.

Classify

A transfer-learned CNN scores all five classes and reports a calibrated confidence.

Explain

A Grad-CAM heatmap shows exactly which regions influenced the prediction.
